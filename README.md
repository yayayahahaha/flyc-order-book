### Environment

| App    | Version |
| ------ | ------- |
| NodeJs | 18      |
| Pnpm   | 8.12.1  |

### How To Start

```shell
pnpm install
pnpm dev
```

Then visit [http://localhost:7777/flyc-order-book/](http://localhost:7777/flyc-order-book/)
